

# 3.3.3 Nissaraṇiyasutta




(72.)

465\. Vuttañhetaṃ bhagavatā vuttamarahatāti me sutaṃ—

466\. “Tisso imā, bhikkhave, nissaraṇiyā dhātuyo. Katamā tisso? Kāmānametaṃ nissaraṇaṃ yadidaṃ nekkhammaṃ, rūpānametaṃ nissaraṇaṃ yadidaṃ āruppaṃ, yaṃ kho pana kiñci bhūtaṃ saṅkhataṃ paṭiccasamuppannaṃ nirodho tassa nissaraṇaṃ—  imā kho, bhikkhave, tisso nissaraṇiyā dhātuyo”ti. Etamatthaṃ bhagavā avoca. Tatthetaṃ iti vuccati—

467\. _“Kāmanissaraṇaṃ ñatvā,_  
_rūpānañca atikkamaṃ;_  
_Sabbasaṅkhārasamathaṃ,_  
_phusaṃ ātāpi sabbadā._  


468\. _Sa ve sammaddaso bhikkhu,_  
_yato tattha vimuccati;_  
_Abhiññāvosito santo,_  
_sa ve yogātigo munī”ti._  


469\. Ayampi attho vutto bhagavatā, iti me sutanti.

---

470\. Tatiyaṃ.
