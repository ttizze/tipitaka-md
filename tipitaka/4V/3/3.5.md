# 3.5 Mānattasata

(166.)

697\. Idha pana, bhikkhave, bhikkhu sambahulā saṃghādisesā āpattiyo āpajjitvā appaṭicchādetvā vibbhamati. So puna upasampanno tā āpattiyo nacchādeti. Tassa, bhikkhave, bhikkhuno mānattaṃ dātabbaṃ. (1)
