

# 3.3.11 Ekādasamasutta




(133.)

1102\. “Dasayime, bhikkhave, sammattā. Katame dasa? Sammādiṭṭhi, sammāsaṅkappo, sammāvācā, sammākammanto, sammāājīvo, sammāvāyāmo, sammāsati, sammāsamādhi, sammāñāṇaṃ, sammāvimutti—  ime kho, bhikkhave, dasa sammattā”ti.

1103\. Ekādasamaṃ.

1104\. Parisuddhavaggo tatiyo.
