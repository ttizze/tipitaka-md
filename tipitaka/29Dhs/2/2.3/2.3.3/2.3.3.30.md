

# 2.3.3.30 Satisampajaññaduka





(1358.)

2063\. Tattha katamā sati? Yā sati anussati paṭissati sati saraṇatā dhāraṇatā apilāpanatā asammussanatā sati satindriyaṃ satibalaṃ sammāsati—  ayaṃ vuccati sati.

(1359.)

2064\. Tattha katamaṃ sampajaññaṃ? Yā paññā pajānanā…pe…  amoho dhammavicayo sammādiṭṭhi—  idaṃ vuccati sampajaññaṃ.
