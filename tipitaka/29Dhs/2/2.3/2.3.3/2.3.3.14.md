

# 2.3.3.14 Pubbantānudiṭṭhiduka





(1326.)

2031\. Tattha katamā pubbantānudiṭṭhi? Pubbantaṃ ārabbha yā uppajjati diṭṭhi diṭṭhigataṃ…pe…  vipariyāsaggāho—  ayaṃ vuccati pubbantānudiṭṭhi.

(1327.)

2032\. Tattha katamā aparantānudiṭṭhi? Aparantaṃ ārabbha yā uppajjati diṭṭhi diṭṭhigataṃ…pe…  vipariyāsaggāho—  ayaṃ vuccati aparantānudiṭṭhi.
