

# 3.3.9 Parikuppasutta




(129.)

725\. “Pañcime, bhikkhave, āpāyikā nerayikā parikuppā atekicchā. Katame pañca? Mātā jīvitā voropitā hoti, pitā jīvitā voropito hoti, arahaṃ jīvitā voropito hoti, tathāgatassa duṭṭhena cittena lohitaṃ uppāditaṃ hoti, saṃgho bhinno hoti. Ime kho, bhikkhave, pañca āpāyikā nerayikā parikuppā atekicchā”ti.
