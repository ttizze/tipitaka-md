

# 3.3.4 Ekasutta




(307.)

1305\. “Ekena ca kho, bhikkhave, balena samannāgato puriso mātugāmaṃ abhibhuyya vattati. Katamena ekena balena? Issariyabalena abhibhūtaṃ mātugāmaṃ neva rūpabalaṃ tāyati, na bhogabalaṃ tāyati, na ñātibalaṃ tāyati, na puttabalaṃ tāyati, na sīlabalaṃ tāyatī”ti.
