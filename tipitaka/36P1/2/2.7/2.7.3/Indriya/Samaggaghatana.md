# Samaggaghaṭanā

(462.)

1481\. Indriya sahajāta nissaya magga atthi avigatanti nahetuyā satta, naārammaṇe satta…pe…  naaññamaññe tīṇi, naupanissaye satta…pe…  nasampayutte tīṇi, navippayutte tīṇi, nonatthiyā satta, novigate satta.

1482\. Indriya sahajāta aññamañña nissaya magga atthi avigatanti nahetuyā tīṇi…pe…  nasampayutte ekaṃ, navippayutte tīṇi, nonatthiyā tīṇi, novigate tīṇi.

1483\. Indriya sahajāta aññamañña nissaya magga sampayutta atthi avigatanti nahetuyā tīṇi…pe…  novigate tīṇi.

1484\. Indriya sahajāta nissaya magga vippayutta atthi avigatanti nahetuyā tīṇi…pe…  novigate tīṇi. [Avipāka—  4]

1485\. Indriya sahajāta nissaya vipāka magga atthi avigatanti nahetuyā ekaṃ…pe…  novigate ekaṃ.

1486\. Indriya sahajāta aññamañña nissaya vipāka magga atthi avigatanti nahetuyā ekaṃ…pe…  novigate ekaṃ.

1487\. Indriya sahajāta aññamañña nissaya vipāka magga sampayutta atthi avigatanti nahetuyā ekaṃ…pe…  novigate ekaṃ.

1488\. Indriya sahajāta nissaya vipāka magga vippayutta atthi avigatanti nahetuyā ekaṃ…pe…  novigate ekaṃ.

1489\. Indriya sahajāta aññamañña nissaya vipāka magga vippayutta atthi avigatanti nahetuyā ekaṃ…pe…  novigate ekaṃ. [Savipāka—  5]
